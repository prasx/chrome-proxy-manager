# 🛡️ Proxy Manager

Chrome расширение для управления маршрутизацией трафика через локальный Xray/V2Ray прокси.

## Что это?

Расширение позволяет гибко управлять тем, какие сайты открываются через прокси, а какие напрямую. Использует PAC (Proxy Auto-Config) для автоматической маршрутизации.

## Возможности

- **Два независимых списка:**
  - 🔒 Через прокси - сайты которые идут через прокси
  - ✅ Напрямую - сайты которые всегда открываются напрямую (приоритет выше)

- **Поддержка паттернов:**
  - `google.com` - точный домен и все поддомены
  - `*.ru` - все сайты в зоне .ru
  - `*.google.com` - все поддомены Google

- **Несколько прокси серверов:**
  - Добавляйте несколько прокси с fallback
  - SOCKS5, SOCKS4, HTTP, HTTPS
  - Переключение между серверами

- **Импорт списков:**
  - Загрузка списков с URL (например, списки заблокированных сайтов)
  - Быстрые ссылки на популярные списки
  - Экспорт/импорт конфигурации

- **Логи маршрутизации:**
  - Таблица с историей запросов
  - Фильтры по типу (PROXY/DIRECT)
  - Видно какое правило сработало

## Установка

1. Скачайте все файлы расширения
2. Откройте Chrome → `chrome://extensions/`
3. Включите "Режим разработчика" (Developer mode)
4. Нажмите "Загрузить распакованное расширение" (Load unpacked)
5. Выберите папку с файлами

## Настройка

### 1. Добавьте прокси сервер

В разделе "Прокси серверы":
- Укажите адрес (например, `127.0.0.1` или `192.168.1.100`)
- Укажите порт (например, `1080`)
- Выберите тип (SOCKS5 рекомендуется)

### 2. Добавьте сайты

**Вкладка "🔒 Через прокси":**
- Добавьте сайты которые должны открываться через прокси
- Примеры: `google.com`, `*.ru`, `youtube.com`

**Вкладка "✅ Напрямую":**
- Добавьте сайты которые должны открываться напрямую
- Примеры: `localhost`, `127.0.0.1`, `*.local`
- Можно загрузить готовый список по кнопке "🇷🇺 RU Whitelist"

### 3. Проверьте работу

Нажмите кнопку "🔍 Тест подключения" - покажет ваш IP через прокси.

## Логика работы

1. Проверяется список "Напрямую" → если совпадает = DIRECT
2. Проверяется список "Через прокси" → если совпадает = PROXY
3. Если ничего не совпало = DIRECT (по умолчанию)

## Примеры использования

### Пример 1: Обход блокировок

**Через прокси:**
- `*.ru` 
- `google.com`
- `youtube.com`
- `*.company.com`

**Напрямую:**
- `localhost`
- `127.0.0.1`
- Загрузить список российских сайтов по кнопке

### Экспорт конфигурации

Кнопка 💾 в верхнем меню → сохраняет:
- Все прокси серверы
- Список "Через прокси"
- Список "Напрямую"

### Импорт конфигурации

Кнопка 📂 в верхнем меню → загружает полную конфигурацию.

### Проверка логов

Откройте вкладку "📋 Логи" в расширении - там видно:
- Какие сайты через прокси
- Какие напрямую
- Какое правило сработало
