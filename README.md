# 🛡️ Proxy Manager

> Chrome расширение для управления маршрутизацией трафика через локальный Xray/V2Ray прокси

---

## ✨ Возможности

### 🎯 Основные функции
- **Два списка доменов** — через прокси и напрямую (с приоритетом)
- **Wildcard паттерны** — `google.com`, `*.ru`, `*.google.com`
- **Несколько прокси** — SOCKS5, SOCKS4, HTTP, HTTPS с автоматическим fallback
- **Импорт/экспорт** — списков и полной конфигурации
- **Загрузка списков** — импорт доменов по URL

### 🔍 Анализ и автоматизация
- **Анализ страницы** — автоматическое обнаружение связанных доменов (CDN, зеркала)
- **Автодобавление** — связанные домены добавляются автоматически
- **Контекстное меню** — быстрые действия через ПКМ в браузере
- **Уведомления** — информирование о всех действиях

### 📊 Мониторинг
- **Статистика** — счётчики запросов через прокси и напрямую
- **График активности** — визуализация последнего часа
- **Топ-10 доменов** — самые посещаемые сайты
- **Логи маршрутизации** — детальная история с фильтрами
- **Автоочистка** — логи очищаются каждые 3 часа

### ⚙️ Настройки
- **Тест подключения** — проверка IP через прокси
- **Управление прокси** — добавление, редактирование, переключение
- **Включение/отключение** — для каждого домена и прокси отдельно

---

## 🚀 Установка

1. Откройте `chrome://extensions/`
2. Включите **"Режим разработчика"**
3. **"Загрузить распакованное расширение"** → выберите папку

---

## 💡 Быстрый старт

```
1. Добавьте прокси: 127.0.0.1:1080 (SOCKS5)
2. Добавьте домены в список "Через прокси"
3. ПКМ на странице → "Добавить в список прокси"
4. Нажмите "🔍 Анализ страницы" → добавьте связанные домены
5. Проверьте: "🔍 Тест подключения"
```

---

## 🎨 Интерфейс

### Вкладки
- **⚙️ Настройки** — управление прокси и списками доменов
- **📊 Статистика** — счётчики, график активности, топ доменов
- **📋 Логи** — история маршрутизации с фильтрами

### Контекстное меню (ПКМ)
- 🔒 Добавить в список прокси
- ✅ Добавить в список напрямую
- 🔍 Анализ связанных доменов
- 🗑️ Удалить из всех списков

---

## 📖 Логика работы

```
1. Проверка списка "Напрямую" → DIRECT
2. Проверка списка "Через прокси" → PROXY
3. Не совпало → DIRECT (по умолчанию)
```
