# 🛡️ Proxy Manager

Chrome расширение для управления маршрутизацией трафика через локальный Xray/V2Ray прокси.

## Возможности

- **Два списка:** через прокси и напрямую (приоритет выше)
- **Паттерны:** `google.com`, `*.ru`, `*.google.com`
- **Несколько прокси:** SOCKS5, SOCKS4, HTTP, HTTPS с fallback
- **Импорт/экспорт** списков и конфигурации
- **Логи** маршрутизации с фильтрами

## Установка

1. Откройте `chrome://extensions/`
2. Включите "Режим разработчика"
3. "Загрузить распакованное расширение" → выберите папку

## Настройка

1. **Добавьте прокси:** адрес (например, `127.0.0.1`), порт (`1080`), тип (SOCKS5)
2. **Добавьте сайты:**
   - 🔒 Через прокси: `google.com`, `*.ru`
   - ✅ Напрямую: `localhost`, `*.local` или загрузите готовый список
3. **Проверьте:** кнопка "🔍 Тест подключения"

## Логика работы

1. Список "Напрямую" → DIRECT
2. Список "Через прокси" → PROXY
3. Не совпало → DIRECT

## Пример использования

**Через прокси:** `*.ru`, `google.com`, `youtube.com`  
**Напрямую:** `localhost`, `127.0.0.1` + загрузить список российских сайтов

**Экспорт/импорт:** кнопки 💾 и 📂 в верхнем меню  
**Логи:** вкладка "📋 Логи" показывает историю и правила маршрутизации
